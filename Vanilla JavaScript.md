    - Backend
        - Python

- Frontend

- JavaScript Version
    - ECMAScript는 JS의 Specification
    - 브라우져의 작동방식을 기술
- Vanilla JavaScript
    - 라이브러리가 없는 순수 JavaScript
    - 브라우저를 통해 제공된 JavaScript
    - 웹에서의 기초가 되는 언어
